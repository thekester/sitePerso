Politique de confidentialité

1- Le type d'information récolté

Mon application récolte les publications sur la page dans le but de les afficher uniquement en les stockent dans une pile qui une fois affichée disparaît

2- Comment ces informations sont récoltés

Grace à une application que j'ai créé sur facebook for developpers avec l'autorisation pages_read_user_content me permettant ainsi de lire le contenu de l'utilisateur généré sur la page afin de me permettre de récupérer et d'afficher les posts sur la page

3- Comment ces informations sont stockées et protégées

Je stocke uniquement l'information(publication) dans une pile que j'affiche puis la pile disparait
